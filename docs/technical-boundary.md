# Technical Boundary

This repository is intentionally limited to public, sanitized evidence records.

## Included

- Public capability summary.
- 2026-07-08 board test result.
- Aggregated metrics.
- Boundary notes.

## Not Included

- Source code.
- RTL.
- Bitstream or firmware.
- Programming files.
- Private decision rules.
- Device identifiers.
- Local machine paths.
- Customer data.
- Keys or authentication material.
- Production deployment details.

## Current Validation Scope

The current public record supports the following statement:

```text
G1 has completed a controlled board-side evidence-frame validation for action-before-execution routing, decision, repair/review flow, and evidence-frame rendering.
```

It does not claim:

- Production deployment.
- Industrial safety certification.
- Replacement of existing controllers.
- Real customer device integration.
- Control of a production actuator in this public test.
